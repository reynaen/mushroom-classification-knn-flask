About This App

K-Nearest Neighbors for Mushroom Classification

About the Algorithm

This application uses the K-Nearest Neighbors (KNN) algorithm to classify mushrooms as edible or poisonous based on 22 physical characteristics.

  • Algorithm: KNN (k=5)
  • Distance Metric: Euclidean
  • Library: Scikit-Learn
  • Model Accuracy: {{ accuracy }}%
About the Dataset

The dataset is the UCI Mushroom Dataset, which contains 8,124 mushroom samples with 22 categorical features.

  • Total Samples: 8,124
  • Edible: 4,208 samples
  • Poisonous: 3,916 samples
  • Features: 22 categorical attributes
